# SPEC.md — design-widget-schedule 개발 스펙

## 프로젝트 정보
- **레포:** `Peekaboo325/design-widget-schedule`
- **프레임워크:** Electron + React + Vite
- **용도:** IMC 3본부 디자인팀 팀원용 바탕화면 위젯

---

## 위젯 공통

### 기본 기능
- 오늘 날짜/요일 표시 (위젯 상단, 예: "5월 19일 (월)")
- 5분 자동 새로고침 + 수동 새로고침 버튼
- 프레임 없는 창, 드래그 이동 가능

### 위젯 설정
- 항상 위에 고정 / 해제 토글
- 투명도 조절 (슬라이더, 0~1)
- 테마 컬러 커스텀 (컬러피커 또는 프리셋)
- 크기 전환 (S / M / L)

### 팀원 식별
- 최초 실행 시 드롭다운에서 본인 이름 선택
- 팀원 목록은 GAS API (`?type=members`)에서 동적 로드
- 선택값은 로컬에 저장, 설정에서 변경 가능

---

## 탭 1: 스케줄 / 공유대기

한 화면에 구분선으로 분리 (탭 내 전환 없음, L 사이즈 기준).

### 스케줄 영역
- 미정 / 대기 / 진행 상태의 작업
- 컬럼: 광고주 / 비고 / 수량 / 상태

### 공유대기 영역
- 완료했지만 메일 미발송(공유 미체크) 건
- 컬럼: 광고주 / 비고 / 수량

### 크기별 정보 단계
| 크기 | 스케줄 | 공유대기 |
|------|--------|----------|
| S | 미처리 건수 숫자만 | 건수 뱃지 ("N건 미발송") |
| M | 광고주별 건수 요약 | 건수 뱃지 |
| L | 전체 테이블 | 전체 목록 (구분선 아래) |

---

## 탭 2: 기획 해석 및 실행 점검 가이드

## GAS API — schedule-widget-api.gs

스케줄러 스프레드시트(3본부 디자인팀 업무시트) 내 바인딩 스크립트.

### 엔드포인트

#### `?type=members`
팀원 목록 반환.
```json
{ "members": ["부수빈", "이소빈", "조희주", "강진이", "김수현", "서아라"] }
```

#### `?type=schedule&member=조희주`
해당 팀원의 미완료 작업 + 공유대기 반환.
```json
{
  "schedule": [
    { "광고주": "하프클리닉", "비고": "메타07", "수량": 1, "상태": "대기" },
    { "광고주": "침향환", "비고": "팝업배너01", "수량": 1, "상태": "진행" }
  ],
  "pending": [
    { "광고주": "사군자", "비고": "DB배너01", "수량": 1 }
  ],
  "summary": {
    "total": 2,
    "pending": 1
  }
}
```

### 참조 시트/컬럼
- 시트: `💛신규·유지보수` (9행 헤더, 10행부터 데이터)
- E(광고주), F(작업자), I(수량), J(비고), K(상태), L(공유)

### 팀원 관리
- 스크립트 내 `MEMBERS` 배열에 이름 추가/삭제
- 위젯 재빌드 불필요 (API에서 동적 로드)

## 개발 순서
1. Electron + React + Vite 셋업 (프레임 없는 투명 창, 드래그, 항상 위에)
2. 위젯 공통 설정 UI (투명도, 컬러, 크기 전환, 고정 토글)
3. GAS API 연결 (schedule-widget-api 배포 확인)
4. 스케줄/공유대기 뷰 (S/M/L 단계별)
5. 가이드 체크리스트 탭
6. 팀원 선택 + 로컬 저장

# SUIT 폰트 파일 위치

여기에 **`SUIT-Variable.woff2`** 파일을 두세요.

## 다운로드
- 가변 폰트 1개 파일로 weight 100~900 전부 커버
- 다운로드 URL: https://cdn.jsdelivr.net/gh/sun-typeface/SUIT@2/fonts/variable/woff2/SUIT-Variable.woff2
- 또는 GitHub: https://github.com/sun-typeface/SUIT/tree/main/fonts/variable/woff2

## 파일이 없을 때
앱은 정상 실행되며 시스템 폰트(Pretendard, Apple SD Gothic Neo 등)로 fallback.

## 파일 두는 방법
1. 위 URL에서 `SUIT-Variable.woff2` 다운로드
2. 이 디렉토리(`src/assets/fonts/`)에 저장
3. `git add src/assets/fonts/SUIT-Variable.woff2 && git commit -m "폰트 추가" && git push`
